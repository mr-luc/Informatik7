# Informatik 7 · Interaktive Arbeitsblätter

Dieses Repository enthält interaktive Arbeitsblätter für den Informatikunterricht in Klasse 7.

## Arbeitsblätter

### AB 3 · Die drei Grundstrukturen

Thema: Sequenz, Selektion, Iteration

Pfad im Repository:

```text
ab3-grundstrukturen/index.html
```

Funktionen:

- interaktive Eingabefelder
- Sofortkontrolle
- Lösung einblendbar
- Fortschrittsanzeige
- Druckmodus
- Speicherung im Browser über `localStorage`
- geeignet für iPad, PC und Beamer

## Startseite

Die Datei `index.html` ist die Startseite für GitHub Pages und verlinkt auf die interaktiven Arbeitsblätter.

Wenn GitHub Pages aktiviert ist, sollte die Seite hier erreichbar sein:

```text
https://mr-luc.github.io/Informatik7/
```

Direkter Link zu AB 3:

```text
https://mr-luc.github.io/Informatik7/ab3-grundstrukturen/
```

## Hinweis

Falls GitHub Pages noch nicht aktiv ist:

1. GitHub-Repository öffnen
2. Settings → Pages
3. Source: `Deploy from a branch`
4. Branch: `main`
5. Folder: `/root`
6. Save

Nach kurzer Zeit ist die Seite online.

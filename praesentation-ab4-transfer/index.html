A2Transferfrage

Ab wann ist die Summe groß?

Gesucht ist der kleinste Wert für n, bei dem gilt:

1 + 2 + 3 + … + n > 100
Tipp: Probiere n = 10, 12, 14 …

Lösung: n = 14, denn 1 + 2 + … + 14 = 105.

Formel: n(n+1)/29
A3Bonusaufgabe
A3 · 8 P.

PIN-Check mit 3 Versuchen

  1. Nach der PIN fragen
  2. Bei richtiger PIN: Zugang gewährt
  3. Bei falscher PIN: Versuche hochzählen
  4. Nach 3 falschen Versuchen: System gesperrt
Baue SOLANGE und WENN ein10
A3Beispiellösung

PIN-Check: mögliche Lösung

Versuche = 0 SOLANGE Versuche < 3 WIEDERHOLE EINGABE: PIN WENN PIN = "1234" DANN AUSGABE: "Zugang gewährt!" ENDE SONST Versuche = Versuche + 1 ENDE WENN ENDE SOLANGE AUSGABE: "System gesperrt!"

Struktur 1: Iteration → SOLANGE · Struktur 2: Selektion → WENN

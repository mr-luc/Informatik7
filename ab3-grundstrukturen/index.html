Sequenz (S)

Schritte nacheinander abarbeiten.

Kein besonderes Schlüsselwort
Licht an.
Tasche nehmen.
Tür öffnen.
Form: gerade Linie ↓

Selektion (Se)

Eine Entscheidung — JA oder NEIN.

WENN ... DANN (... SONST)
WENN Regen DANN
   Schirm nehmen
SONST
   kein Schirm
ENDE WENN
Form: Verzweigung

Iteration (I)

Etwas wiederholen, solange eine Bedingung gilt.

SOLANGE ... WIEDERHOLE
SOLANGE Hunger WIEDERHOLE
   Bissen essen
ENDE SOLANGE
Form: Schleife

Pseudocode-BeispielSchlüsselwortS / Se / I

2: WENN ... DANN ... SONST → Se
3: SOLANGE ... WIEDERHOLE → I
4: kein besonderes Schlüsselwort → S
5: WENN ... DANN ... SONST → Se
6: WIEDERHOLE → I
